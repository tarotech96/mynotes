# My Notes App

## List commands

- To add a package

    ```javascript
      flutter pub add {packgage_name}
    ```

- To get package

  ```javascript
      flutter pub get
  ```

- To run app

  ```javascript
      flutter run
  ```

- To clean package

  ```javascript
      flutter clean
  ```
